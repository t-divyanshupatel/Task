# addPolicies

This method adds policies to an RBAC role. It sends a request to the
Add Role Policies API route.

## Example

```ts
sdk.admin.rbacRole.addPolicies("role_123", {
  policies: ["policy_123", "policy_456"]
})
.then(({ policies }) => {
  console.log(policies)
})
```
