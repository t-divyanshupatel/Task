# AuthMfaStatus

 **AuthMfaStatus**: `"pending"` \| `"enabled"` \| `"disabled"`

Status of a multi-factor authentication (MFA) setup for an authentication identity.

:::note

This is available starting from [Medusa v2.15.3](https://github.com/medusajs/medusa/releases/tag/v2.15.3).

:::
