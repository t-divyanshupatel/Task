# cancelValidateOrder - Medusa Core Workflows Reference

This documentation provides a reference to the `cancelValidateOrder`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that an order can be canceled. If the order has fulfillments that
aren't canceled, or the order was canceled previously, the step throws an error.

:::note

You can retrieve an order's details using [Query](https://docs.medusajs.com/learn/fundamentals/module-links/query),
or [useQueryGraphStep](https://docs.medusajs.com/resources/references/medusa-workflows/steps/useQueryGraphStep).

:::

## Example

```ts
const data = cancelValidateOrder({
  order: {
    id: "order_123",
    // other order details...
  },
  input: {
    order_id: "order_123",
  }
})
```
