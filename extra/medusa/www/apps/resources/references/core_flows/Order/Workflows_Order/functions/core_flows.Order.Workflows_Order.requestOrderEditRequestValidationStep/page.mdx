# requestOrderEditRequestValidationStep - Medusa Core Workflows Reference

This documentation provides a reference to the `requestOrderEditRequestValidationStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that a order edit can be requested.
If the order is canceled or the order change is not active, the step will throw an error.

:::note

You can retrieve an order and order change details using [Query](https://docs.medusajs.com/learn/fundamentals/module-links/query),
or [useQueryGraphStep](https://docs.medusajs.com/resources/references/medusa-workflows/steps/useQueryGraphStep).

:::

## Example

```ts
const data = requestOrderEditRequestValidationStep({
  order: {
    id: "order_123",
    // other order details...
  },
  orderChange: {
    id: "orch_123",
    // other order change details...
  },
})
```
