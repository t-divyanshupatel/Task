# releaseLockStep - Medusa Core Workflows Reference

This documentation provides a reference to the `releaseLockStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step releases a lock for a given key. Learn more about locks in the [Locking Module](https://docs.medusajs.com/resources/infrastructure-modules/locking)
guide.

## Example

```ts
const data = releaseLockStep({
  key: "my-lock-key"
})
```
