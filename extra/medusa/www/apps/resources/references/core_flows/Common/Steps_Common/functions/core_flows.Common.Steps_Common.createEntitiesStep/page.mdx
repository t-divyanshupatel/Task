# createEntitiesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createEntitiesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates one or more entities using methods in a module's service.

  moduleRegistrationName: Modules.CART,
