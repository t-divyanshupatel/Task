# confirmInventoryStep - Medusa Core Workflows Reference

This documentation provides a reference to the `confirmInventoryStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that items in the cart have sufficient inventory quantity.
If an item doesn't have sufficient inventory, an error is thrown.

## Example

```ts
confirmInventoryStep({
  items: [
    {
      inventory_item_id: "iitem_123",
      required_quantity: 1,
      allow_backorder: false,
      quantity: 1,
      location_ids: ["sloc_123"]
    }
  ]
})
```
