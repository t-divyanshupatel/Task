# validateCustomerStoreCreditAccountStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateCustomerStoreCreditAccountStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that a customer's store credit account exists and has a positive
balance. It throws an error if the account is not found or has no balance.

## Example

```ts
const data = validateCustomerStoreCreditAccountStep({
  storeCreditAccount: {
    id: "sca_123",
    balance: 100,
    // other store credit account properties...
  },
})
```
