# createLineItemAdjustmentsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createLineItemAdjustmentsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates line item adjustments in a cart, such as when a promotion is applied.

## Example

```ts
createLineItemAdjustmentsStep({
  lineItemAdjustmentsToCreate: [
    {
      item_id: "litem_123",
      code: "10OFF",
      amount: 10,
    }
  ]
})
```
