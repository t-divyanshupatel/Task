# validateStoreCreditAccountStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateStoreCreditAccountStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that all gift cards on the cart have a positive balance.
It throws an error for any gift card with zero or negative balance.

## Example

```ts
const data = validateStoreCreditAccountStep({
  giftCards: [
    {
      id: "gc_123",
      code: "GC-XXXX",
      // other gift card properties...
    }
  ],
  giftCardsBalanceMap: {
    "GC-XXXX": {
      balance: 100,
      credit: 150,
      debit: 50,
    }
  },
})
```
