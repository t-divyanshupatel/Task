# validateCartItemsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateCartItemsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that a cart has at least one line item before
completing the cart and placing an order. If the cart has no items,
the step throws an error.

:::note

This is available starting from [Medusa v2.14.3](https://github.com/medusajs/medusa/releases/tag/v2.14.3).

:::

## Example

```ts
validateCartItemsStep({
  cart
})
```
