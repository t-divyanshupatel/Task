# setTaxLinesForItemsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `setTaxLinesForItemsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step sets the tax lines of shipping methods and line items in a cart.

:::tip

You can use the [retrieveCartStep](../core_flows.Cart.Steps_Cart.retrieveCartStep/page.mdx) to retrieve a cart's details.

:::

## Example

```ts
const data = setTaxLinesForItemsStep({
  // retrieve the details of the cart from another workflow
  // or in another step using the Cart Module's service
  cart,
  "item_tax_lines": [{
    "rate": 48,
    "code": "CODE123",
    "name": "Tax rate 2",
    "provider_id": "provider_1",
    "line_item_id": "litem_123"
  }],
  "shipping_tax_lines": [{
    "rate": 49,
    "code": "CODE456",
    "name": "Tax rate 1",
    "provider_id": "provider_1",
    "shipping_line_id": "sm_123"
  }]
})
```
