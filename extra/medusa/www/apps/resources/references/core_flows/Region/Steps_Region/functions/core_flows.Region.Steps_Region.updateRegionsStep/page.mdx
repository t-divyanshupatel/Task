# updateRegionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateRegionsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates regions matching the specified filters.

## Example

```ts
const data = updateRegionsStep({
  selector: {
    id: "reg_123"
  },
  update: {
    name: "United States"
  }
})
```
