# validateVariantPriceLinksStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateVariantPriceLinksStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that the specified variants have prices.
If not valid, the step throws an error.

## Example

```ts
const data = validateVariantPriceLinksStep([
  {
    prices: [
      {
        variant_id: "variant_123",
      }
    ]
  }
])
```
