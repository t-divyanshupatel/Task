# validateInventoryItemsForCreate - Medusa Core Workflows Reference

This documentation provides a reference to the `validateInventoryItemsForCreate`. It belongs to the `@medusajs/medusa/core-flows` package.

This step checks whether a variant already has an inventory item.
If so, the step will throw an error.

## Example

```ts
const data = validateInventoryItemsForCreate([
  {
    tag: "variant_123"
  }
])
```
