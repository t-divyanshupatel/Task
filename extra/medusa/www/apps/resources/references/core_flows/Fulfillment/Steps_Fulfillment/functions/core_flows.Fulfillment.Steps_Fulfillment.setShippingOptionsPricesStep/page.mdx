# setShippingOptionsPricesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `setShippingOptionsPricesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step sets the prices of one or more shipping options.

## Example

```ts
const data = setShippingOptionsPricesStep([
  {
    id: "so_123",
    prices: [
      {
        amount: 1000,
        currency_code: "usd",
      }
    ]
  }
])
```
