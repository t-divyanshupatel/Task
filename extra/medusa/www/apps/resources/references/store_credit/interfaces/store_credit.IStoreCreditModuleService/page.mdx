# IStoreCreditModuleService Reference

This section of the documentation provides a reference to the `IStoreCreditModuleService` interface’s methods. This is the interface developers use to use the functionalities provided by the Store Credit Module.

<Note>

You should only use the methods in this reference when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>
