# retrieveAccountStats - Store Credit Module Reference

This documentation provides a reference to the `retrieveAccountStats` method. This belongs to the Store Credit Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method retrieves the balance statistics for a store credit account.

## Example

```ts
const stats = await storeCreditModuleService.retrieveAccountStats({
  account_id: "sca_123",
})
```
