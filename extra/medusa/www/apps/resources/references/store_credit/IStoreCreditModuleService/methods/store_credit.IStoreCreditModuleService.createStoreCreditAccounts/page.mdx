# createStoreCreditAccounts - Store Credit Module Reference

This documentation provides a reference to the `createStoreCreditAccounts` method. This belongs to the Store Credit Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## createStoreCreditAccounts(data, sharedContext?): Promise&#60;[ModuleStoreCreditAccount](../../../types/store_credit.ModuleStoreCreditAccount/page.mdx)&#62;

This method creates a store credit account.

### Example

```ts
const account = await storeCreditModuleService.createStoreCreditAccounts({
  currency_code: "usd",
  customer_id: "cust_123",
})
```
