# creditAccounts - Store Credit Module Reference

This documentation provides a reference to the `creditAccounts` method. This belongs to the Store Credit Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method credits one or more store credit accounts by creating credit transactions.

## Example

```ts
const transactions = await storeCreditModuleService.creditAccounts([
  {
    account_id: "sca_123",
    amount: 100,
    reference: "order",
    reference_id: "order_123",
  },
])
```
