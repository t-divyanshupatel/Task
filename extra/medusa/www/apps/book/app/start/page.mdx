# Set Up Medusa with AI Agents

Create an ecommerce store with Medusa and go live in minutes. This page is designed for AI agents and humans who want to get started quickly with an AI agent.

If you're a human, copy this prompt and run it in your AI agent:

```bash title="Prompt"
Fetch https://docs.medusajs.com/start and create an ecommerce store with Medusa Cloud
```

The AI agent will set up Medusa locally, deploy it to [Cloud](!cloud!), and help you build your custom commerce features.

## FAQ

### What does the prompt do?

The prompt instructs the AI agent to fetch the content of this page, which results in a prompt with step-by-step instructions to set up Medusa.

The AI agent will then execute the instructions, which include installing Medusa, deploying it to [Cloud](!cloud!), and providing next steps to build custom commerce features.

### Do I need a Cloud account?

No, the AI agent will set up Medusa locally on your machine. However, to deploy the store, the AI agent will guide you to create a [Cloud](!cloud!) account if you don't have one, and help you with the deployment process.

### Can my AI agent build custom features?

Yes, after setting up Medusa, your AI agent can help you build custom commerce features.

You can instruct your AI agent to implement specific functionalities, integrate third-party services, or build custom workflows using the Medusa Framework.

### Can I use my own domain?

Yes, you can [add custom domains](!cloud!/environments/custom-domains) to your Medusa store deployed on Cloud.
