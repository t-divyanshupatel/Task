# {metadata.title}

[Workflows](!docs!/learn/fundamentals/workflows) are a series of queries and actions, called steps, that complete a task. By using workflows, you benefit from features like data consistency and a reliable roll-back mechanism.

The Store Credit Module provides workflows and steps to manage store credit accounts and their usage in carts. These are provided by the `@medusajs/loyalty-plugin` package.

## Store Credit Account Management Workflows

### Create Store Credit Accounts

Create new store credit accounts for customers:

```ts
import { createStoreCreditAccountsWorkflow } from "@medusajs/loyalty-plugin"

const { result } = await createStoreCreditAccountsWorkflow(container)
  .run({
    input: {
      customer_id: "cust_123",
      currency_code: "usd",
      code: "SC-XXXX", // optional, auto-generated if not provided
    },
  })
```

### Claim Store Credit Account

Claim a store credit account for a customer:

```ts
import { claimStoreCreditAccountWorkflow } from "@medusajs/loyalty-plugin"

const { result } = await claimStoreCreditAccountWorkflow(container)
  .run({
    input: {
      customer_id: "cust_123",
      code: "SC-XXXX",
    },
  })
```

## Credit and Debit Operations

### Credit Store Credit Account

Add credits to a store credit account:

```ts
import { creditStoreCreditAccountWorkflow } from "@medusajs/loyalty-plugin"

const { result } = await creditStoreCreditAccountWorkflow(container)
  .run({
    input: {
      account_id: "sca_123",
      amount: 100,
      note: "Refund for order #1234",
      reference: "order",
      reference_id: "order_123",
    },
  })
```

### Credit Accounts Workflow

Credit multiple store credit accounts:

```ts
import { creditAccountsWorkflow } from "@medusajs/loyalty-plugin"

const { result } = await creditAccountsWorkflow(container)
  .run({
    input: [
      {
        account_id: "sca_123",
        amount: 100,
        reference: "order",
        reference_id: "order_123",
      },
      {
        account_id: "sca_456",
        amount: 50,
        reference: "refund",
        reference_id: "refund_123",
      },
    ],
  })
```

### Debit Accounts Workflow

Debit multiple store credit accounts:

```ts
import { debitAccountsWorkflow } from "@medusajs/loyalty-plugin"

const { result } = await debitAccountsWorkflow(container)
  .run({
    input: [
      {
        account_id: "sca_123",
        amount: 25,
        reference: "cart",
        reference_id: "cart_123",
      },
    ],
  })
```

## Cart Integration Workflows

### Add Store Credits to Cart

Apply store credits to a cart as a credit line:

```ts
import { addStoreCreditsToCartWorkflow } from "@medusajs/loyalty-plugin"

await addStoreCreditsToCartWorkflow(container)
  .run({
    input: {
      cart_id: "cart_123",
      amount: 50, // optional, uses full balance if not provided
    },
  })
```
