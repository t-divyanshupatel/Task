# useAuthMfaRecoveryCode - Auth Module Reference

This documentation provides a reference to the `useAuthMfaRecoveryCode` method. This belongs to the Auth Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method uses a recovery code for an auth identity.

:::note

This is available starting from [Medusa v2.15.3](https://github.com/medusajs/medusa/releases/tag/v2.15.3).

:::

## Example

```ts
await authModuleService.useAuthMfaRecoveryCode({
  auth_identity_id: "authusr_123",
  code: "recovery123"
})
```
