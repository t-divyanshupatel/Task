# generateAuthMfaRecoveryCodes - Auth Module Reference

This documentation provides a reference to the `generateAuthMfaRecoveryCodes` method. This belongs to the Auth Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method creates a new set of single-use recovery codes for an auth
identity and invalidates any existing recovery codes.

:::note

This is available starting from [Medusa v2.15.3](https://github.com/medusajs/medusa/releases/tag/v2.15.3).

:::

## Example

```ts
const { codes } = await authModuleService.generateAuthMfaRecoveryCodes({
  auth_identity_id: "authusr_123",
  count: 8
})
```
