# listAddresses

This method retrieves a list of customer addresses. It sends a request to the
[List Customer Addresses](https://docs.medusajs.com/api/admin#customers\_getcustomersidaddresses)
API route.

## Example

```ts
sdk.admin.customer.listAddresses("cus_123")
.then(({ addresses }) => {
  console.log(addresses)
})
```
