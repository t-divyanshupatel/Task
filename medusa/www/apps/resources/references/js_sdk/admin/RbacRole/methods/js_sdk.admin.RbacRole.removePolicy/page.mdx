# removePolicy

This method removes a policy from an RBAC role. It sends a request to the
Remove Role Policy API route.

## Example

```ts
sdk.admin.rbacRole.removePolicy("role_123", "policy_456")
.then(({ deleted }) => {
  console.log(deleted)
})
```
