# setRegionsPaymentProvidersStep - Medusa Core Workflows Reference

This documentation provides a reference to the `setRegionsPaymentProvidersStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step sets the payment providers available in regions.

```ts
const data = setRegionsPaymentProvidersStep({
  input: [
    {
      id: "reg_123",
      payment_providers: ["pp_system", "pp_stripe_stripe"]

})
```
