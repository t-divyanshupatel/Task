# getAllProductsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `getAllProductsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step retrieves all products matching a set of filters.

## Example

To retrieve all products:

```ts
const data = getAllProductsStep({
  select: ["*"],
})
```

To retrieve all products matching a filter:

```ts
const data = getAllProductsStep({
  select: ["*"],
  filter: {
    collection_id: "collection_123"
  }
})
