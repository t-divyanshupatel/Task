# processImportChunksStep - Medusa Core Workflows Reference

This documentation provides a reference to the `processImportChunksStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step parses a CSV file holding products to import, returning the products as
objects that can be imported.

## Example

```ts
const data = parseProductCsvStep("products.csv")
```
