# updateProductTypesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateProductTypesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates product types matching the specified filters.

## Example

```ts
const data = updateProductTypesStep({
  selector: {
    id: "popt_123"
  },
  update: {
    value: "clothing"
  }
})
```
