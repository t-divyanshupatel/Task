# validateStepShippingProfileDelete - Medusa Core Workflows Reference

This documentation provides a reference to the `validateStepShippingProfileDelete`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that the shipping profiles to delete are not linked to any products.
Otherwise, an error is thrown.

## Example

```ts
validateStepShippingProfileDelete({
  links: [
    {
      product_id: "product_123",
      shipping_profile_id: "sp_123"
    }
  ]
})
```
