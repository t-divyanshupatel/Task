# validateCarryPromotionsFlagStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateCarryPromotionsFlagStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that the order change is an exchange and validates that
the promotion allocation is valid for carrying over promotions.

:::note

You can retrieve details of the order and order change using [Query](https://docs.medusajs.com/learn/fundamentals/module-links/query),
or [useQueryGraphStep](https://docs.medusajs.com/resources/references/medusa-workflows/steps/useQueryGraphStep).

:::

:::note

This is available starting from [Medusa v2.12.0](https://github.com/medusajs/medusa/releases/tag/v2.12.0).

:::

## Example

```ts
const data = validateCarryPromotionsFlagStep({
  order: {
    id: "order_123",
    // other order details...
  },
  orderChange: {
    id: "orch_123",
    // other order change details...
  },
  input: {
    order_change_id: "orch_123",
    carry_over_promotions: true,
  }
})
```
