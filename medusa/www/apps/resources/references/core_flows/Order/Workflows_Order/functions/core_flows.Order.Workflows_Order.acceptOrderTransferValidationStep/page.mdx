# acceptOrderTransferValidationStep - Medusa Core Workflows Reference

This documentation provides a reference to the `acceptOrderTransferValidationStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that an order transfer can be accepted. If the
order doesn't have an existing transfer request, the step throws an error.

:::note

You can retrieve an order and order change details using [Query](https://docs.medusajs.com/learn/fundamentals/module-links/query),
or [useQueryGraphStep](https://docs.medusajs.com/resources/references/medusa-workflows/steps/useQueryGraphStep).

:::

## Example

```ts
const data = acceptOrderTransferValidationStep({
  token: "sk_123456",
  order: {
    id: "order_123",
    // other order details...
  },
  orderChange: {
    id: "order_change_123",
    // other order change details...
  }
})
```
