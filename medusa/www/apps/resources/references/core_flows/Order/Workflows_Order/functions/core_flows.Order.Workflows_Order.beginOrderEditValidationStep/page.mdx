# beginOrderEditValidationStep - Medusa Core Workflows Reference

This documentation provides a reference to the `beginOrderEditValidationStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that an order-edit can be requested for an order.
If the order is canceled, the step will throw an error.

:::note

You can retrieve an order's details using [Query](https://docs.medusajs.com/learn/fundamentals/module-links/query),
or [useQueryGraphStep](https://docs.medusajs.com/resources/references/medusa-workflows/steps/useQueryGraphStep).

:::

## Example

```ts
const data = beginOrderEditValidationStep({
  order: {
    id: "order_123",
    // other order details...
  }
})
```
