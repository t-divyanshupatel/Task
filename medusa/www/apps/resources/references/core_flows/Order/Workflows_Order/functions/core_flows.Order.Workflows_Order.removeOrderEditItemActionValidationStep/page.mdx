# removeOrderEditItemActionValidationStep - Medusa Core Workflows Reference

This documentation provides a reference to the `removeOrderEditItemActionValidationStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that an item that was added in the order edit can be removed
from the order edit. If the order is canceled or the order change is not active,
the step will throw an error.

:::note

You can retrieve an order and order change details using [Query](https://docs.medusajs.com/learn/fundamentals/module-links/query),
or [useQueryGraphStep](https://docs.medusajs.com/resources/references/medusa-workflows/steps/useQueryGraphStep).

:::

## Example

```ts
const data = removeOrderEditItemActionValidationStep({
  order: {
    id: "order_123",
    // other order details...
  },
  orderChange: {
    id: "orch_123",
    // other order change details...
  },
  input: {
    order_id: "order_123",
    action_id: "orchact_123",
  }
})
```
