# updateApiKeysStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateApiKeysStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates one or more API keys.

## Example

```ts
const data = updateApiKeysStep({
  selector: {
    id: "apk_123"
  },
  update: {
    title: "Storefront"
  }
})
```
