# associateLocationsWithSalesChannelsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `associateLocationsWithSalesChannelsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates links between stock locations and sales channel records.

## Example

```ts
const data = associateLocationsWithSalesChannelsStep({
  links: [
    {
      sales_channel_id: "sc_123",
      location_id: "sloc_123"
    }
  ]
})
```
