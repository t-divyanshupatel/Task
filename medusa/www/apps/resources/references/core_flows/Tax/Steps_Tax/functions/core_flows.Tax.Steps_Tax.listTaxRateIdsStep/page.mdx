# listTaxRateIdsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `listTaxRateIdsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step retrieves the IDs of tax rates matching the specified filters.

## Example

```ts
const data = listTaxRateIdsStep({
  selector: {
    tax_region_id: "txreg_123"
  }
})
```
