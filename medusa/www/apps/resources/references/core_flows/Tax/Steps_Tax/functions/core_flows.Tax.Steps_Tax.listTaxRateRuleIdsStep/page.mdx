# listTaxRateRuleIdsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `listTaxRateRuleIdsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step retrieves the IDs of tax rate rules matching the specified filters.

## Example

```ts
const data = listTaxRateRuleIdsStep({
  selector: {
    tax_rate_id: "txr_123"
  }
})
```
