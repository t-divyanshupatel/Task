# validateCartStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateCartStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates a cart to ensure it exists and is not completed.
If not valid, the step throws an error.

:::tip

You can use the [retrieveCartStep](../core_flows.Cart.Steps_Cart.retrieveCartStep/page.mdx) to retrieve a cart's details.

:::

## Example

```ts
const data = validateCartStep({
  // retrieve the details of the cart from another workflow
  // or in another step using the Cart Module's service
  cart,
})
```
