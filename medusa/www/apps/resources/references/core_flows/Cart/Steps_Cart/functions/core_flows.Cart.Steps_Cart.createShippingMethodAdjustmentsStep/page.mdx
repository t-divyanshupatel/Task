# createShippingMethodAdjustmentsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createShippingMethodAdjustmentsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates shipping method adjustments for a cart.

## Example

```ts
const data = createShippingMethodAdjustmentsStep({
  "shippingMethodAdjustmentsToCreate": [{
    "shipping_method_id": "sm_123",
    "code": "10OFF",
    "amount": 10
  }]
})
```
