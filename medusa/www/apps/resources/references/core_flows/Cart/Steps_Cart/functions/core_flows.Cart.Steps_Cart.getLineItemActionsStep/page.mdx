# getLineItemActionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `getLineItemActionsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step returns lists of cart line items to create or update based on the
provided input.

## Example

```ts
const data = getLineItemActionsStep({
  "id": "cart_123",
  "items": [{
    "title": "Shirt",
    "quantity": 1,
    "unit_price": 50,
    "cart_id": "cart_123",
  }]
})
```
