# validateGiftCardInCartStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateGiftCardInCartStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that a gift card is present in a cart. It throws an error
if the gift card is not found in the cart's gift cards.

## Example

```ts
const data = validateGiftCardInCartStep({
  cart: {
    id: "cart_123",
    gift_cards: [
      {
        id: "gc_123",
        code: "GC-XXXX",
        // other gift card properties...
      }
    ],
    // other cart properties...
  },
  giftCard: { code: "GC-XXXX" },
})
```
