# deleteGiftCardsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `deleteGiftCardsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step deletes gift cards by their IDs. The step supports rollback by
restoring the deleted gift cards.

## Example

```ts
const data = deleteGiftCardsStep({ id: ["gc_123"] })
```
